# Restaurant demo app (WIP)

## Overview

An app where you can check out local cuisine. Search based on names or categories.

## Running the app

To install all the required packages, run:

```shell
pnpm install
```

To run the app in the **development** mode, run:

```shell
pnpm dev
```

To **build** the app, run:

```shell
pnpm build
```

To run the app in the **production** mode, run:

```shell
pnpm start
```

## Linting and formatting

To lint your code, run:

```shell
pnpm lint
```

To format your code, run:

```shell
pnpm format
```

## Usage (WIP)

Search the cuisine by entering the category, name of the dish etc.
Click on the dish of choice and check out its page.
